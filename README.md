# Sales Performance Dashboard & ETL Pipeline

### Project Overview
This project analyzes sales performance for a global superstore to identify profit trends and regional anomalies.

### Tech Stack
* **Python (Pandas):** For cleaning raw data, removing duplicates, and calculating profit margins.
* **SQL:** For validating data integrity and aggregating regional totals.
* **Power BI:** For interactive visualization, drill-down analysis, and KPI tracking.

### Key Features
* **Data Cleaning:** Python script handles missing values and standardizes date formats.
* **Star Schema:** Modeled relationships between Fact (Sales) and Dimension (Region Managers) tables.
* **Interactive Dashboard:** Features dynamic drill-down by Product Category and time-series trend analysis.
